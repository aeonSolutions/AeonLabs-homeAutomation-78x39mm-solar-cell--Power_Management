# 78x39mm solar cell Power Management 
**(5V, 9V 12V DC MAX 9A)**

This PCB electronics is intented to be used with the 78x39mm solar cell pcb. It features 6x independent power solar lines, switch selectable into 3 different DC outputs:Â 
- 5V
- 9V
- and 12V

for a maximum of 9A.
